# Synfony csv parcer

Tested on:
- Ubuntu 18.04
- PHP 7.2.19


## Installation

```
git clone https://github.com/sabresgr/symfonyParcerExample.git
cd symfonyParcerExample
composer install
```

## Settings
 Change DB user and DB password in .env
```
DATABASE_URL=mysql://USER:PASSWORD@127.0.0.1:3306/importTest
```
